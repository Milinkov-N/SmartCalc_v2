SmartCalc v2.0-0 Documentation

SmartCalc - application for calculating mathematical expressions with additionional Credit Calculator module.

Restrictions

  1. The input string for a mathematical expression is no longer than 255 characters.
  2. The maximum value for the abscissa and ordinate axis is 1000000.
  3. The minimum value for plotting is equal to the maximum value, taken with a minus sign.

Installing

To install the application run make install command
